Partidos, verbas eleitorais e orçamento federal

Esta página organiza duas famílias de dados públicos: verbas eleitorais (Fundo Partidário, Fundo Eleitoral, prestação de contas no TSE) e emendas parlamentares divulgadas pelo Portal da Transparência (CGU), incluindo emendas de bancada estadual, próximas do conceito de articulação partidária por UF. A API de dados exige cadastro gratuito de e-mail para obter a chave.

Como relacionar partidos a “verbas”

Verba eleitoral (campanha e partidos)

Repasses do Fundo Partidário e do Fundo Eleitoral, doações e despesas de campanha são regulados pela Justiça Eleitoral e publicados no DivulgaCandContas e em dados abertos do TSE. Não confundir com empenhos do orçamento federal.

Emendas e bancadas (orçamento da União)

No Portal, emenda de bancada é de autoria da bancada estadual no Congresso (interesse da UF). Já as emendas individuais têm como autor deputado ou senador — dá para associar o parlamentar ao partido da época via bases da Câmara/Senado ou TSE. O quadro abaixo sugere siglas frequentes no RJ para cruzar mentalmente com mandatos e com o TSE.

Consultar emendas via API do Portal

A API oficial é api.portaldatransparencia.gov.br; no navegador costuma dar CORS. Em produção, POLITAPP_PORTAL_API_BASE deve ser a URL do Worker (https://….workers.dev/api-de-dados após deploy) ou uma rota no domínio já ligada ao Worker — o hosting estático sozinho devolve 404 HTML em /api-de-dados. Local: npm run dev e http://127.0.0.1:8787/transparencia.html, ou npm run dev:portal-proxy. Chave: campo abaixo, localStorage ou POLITAPP_PORTAL_TRANSPARENCIA_CHAVE.
