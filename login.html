GitHub Pages só serve páginas estáticas: o MySQL fica no backend (Render, Railway, VPS…). Edite js/api-config.js (productionApi) com a URL HTTPS da API e, no servidor, ALLOWED_ORIGINS + SESSION_CROSS_SITE=true. Localmente use npm start e deixe productionApi vazio.

Backend: .env (MySQL, SESSION_SECRET, CORS). Banco: sql/auth-schema.sql. Usuário: npm run create-user -- email senha
